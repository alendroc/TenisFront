# STRYDE — Frontend

Frontend de la tienda de zapatos STRYDE, desarrollado con Vue.js 3 como parte del Proyecto 2 de Programación Web.

## Tecnologías
- Vue.js 3 (Composition API)
- Vue Router 5
- Pinia
- Axios
- Vite

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build para producción

```bash
npm run build
```
